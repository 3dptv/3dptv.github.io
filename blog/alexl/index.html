New, batteries included Windows installer package is created

Submitted by alexl on Fri, 2010-04-30 22:35
  • 3dptv
  • installation
  • windows

We found a way to create a setup.exe file that will install 3D-PTV software without ActiveTcl being pre-installed. So it's a "batteries included" package that will not include the source code, but can be easily deployed on any computer that runs Windows XP (we haven't test it on other platforms). It seems to be possible to run 3D-PTV from the external hard drive or memory card.

Indeed, I have just installed it on my disk-on-key and it RUNS !!!
